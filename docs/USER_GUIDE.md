# 📘 사용자 가이드

## 1. 로그인 및 회원가입
- 로그인 버튼 클릭 → 회원가입 가능

## 2. 퀴즈 진행
- 카테고리 및 난이도 선택(이지, 노말, 하드드) → 문제 30개 출제

## 3. 설정 사용
- 우측 ⚙️ 클릭 → BGM, 효과음 볼륨 조절 가능

## 4. 결과 확인
- '내 기록 조회' 클릭 시 본인 점수 확인

## 5. 사용자 문제 풀기
- 메인 메뉴 → '다른 사용자 문제'
